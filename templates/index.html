Instalación ¶

Connection via http 

http://localhost:4000/

Si desea utilizar algoritmos de firma de clave asimétrica (pública/privada), incluya los asymmetric_crypto requisitos adicionales. 

$ pip install flask-jwt-extended[asymmetric_crypto]

Tenga en cuenta que si está utilizando ZSH (probablemente también otros shells), deberá escapar de los corchetes 

$ pip install flask-jwt-extended\[asymmetric_crypto\]

Si prefiere instalar desde la fuente, puede clonar este repositorio y ejecutar 

$ python setup.py install
